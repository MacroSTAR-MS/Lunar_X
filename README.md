## 概览
>[!Important]
>通知:
>
>鉴于Onebot协议的各种历史遗留问题和Lagrange V2不再支持Onebot和V1不在再维护的情况下，我们做了一个重大抉择：自Lunar X框架0.1.0.145版本起，Onebot协议适配将存档且不再维护，将开发重心移至Milky协议开发,但这并不代表你后续将无法使用Onebot协议支持，您仍然可以在config.json里将Protocol设置为OneBot来使用Onebot支持

Lunar X是一个适配OneBot v11协议的机器人框架，旨在提供一个方便、高效、超强拓展性的机器人运行结构。
